# CodeAlpha_ProjectName

A simple language translation tool built with HTML, CSS, and JavaScript.

## Features
- Enter text and select source / target languages
- Translate text using Microsoft Translator Text API
- Copy translated text with one click
- Speak the translated text using browser speech synthesis

## Setup
1. Open `script.js`.
2. Replace `<YOUR_TRANSLATOR_API_KEY>` and `<YOUR_TRANSLATOR_RESOURCE_REGION>` with your Azure Translator credentials.
3. Open `index.html` in a browser.

## Notes
- This is a frontend demo and requires a valid Translator API key.
- If you want to deploy the app, host the files on GitHub Pages or any static web host.
